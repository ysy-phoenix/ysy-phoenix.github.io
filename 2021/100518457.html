图论chap5习题解答

T1

考虑$K_{2n}$​中任意一个顶点$v$​，有$2n-1$​​种匹配方法，删去$v$及与之相配的顶点，图变为$K_{2(n-1)}$​​，记$A$为匹配个数，则有递推式$A(K_{2n})=(2n-1)A(K_{2(n-1)})$，从而$A(K_{2n})=(2n-1)!!$​。

同理，对于$K_{n,n}$中任意一个顶点$v$，有$n$中匹配方法，删去$v$及与之相配的顶点，图变为$K_{n-1，n-1}$​，则有递推式$A(K_{n,n}=nA(K_{n-1,n-1}))$，从而$A(K_{n,n})=n!$。

T2

假设存在两个不相同的完备匹配$M_1、M_2$，则$M_1\bigoplus M_2\neq \phi$​​，​且$T[M_1\bigoplus M_2]$中每个顶点的度数都是$2$，由例$1.8,\delta(G)\geq2$，有圈，从而与$T$是树矛盾！

T3

若$k$为偶数，则$K_{k+1}$，有奇数个顶点，必然不会有完备匹配，满足。

若$k$为奇数，取$2k-1$​个顶点，奇偶（下标）顶点之间连边，然后$v_1、v_3，\cdots,v_{2k-5}、v_{2k-3}$​之间连边，这样只有$v_{2k-1}$度为$k-1$，其余度皆为$k$，记此图为$G$，然后取$k$个图$G$和一个顶点$v$，将$v$和每个$G$中的$v_{2k-1}$连边即得$k$正则图。而在上述图中，假设存在完备匹配$M$，其在每个$G$中至多有$k-1$​条边，再加上$v$匹配的那一条边，则$|M|\leq k(k-1)+1$，但另一方面$k>1$时，$v=k(2k-1)+1>2(k(k-1)+1)=2|M|$，这与$|M|=v/2$矛盾！

T4

必要性：假设存在完备匹配$M$，则无论第一个人怎么选$v$，第二个人只需要选择$M$中与$v$​相配的顶点，那么第二个人必胜，与第一个人有必胜策略矛盾！

充分性：考虑$G$的一个最大匹配$M$，则一定存在一点$v_0$未被$M$许配，第一个人只需要选择$v_0$，若$v_0$是孤立点，直接胜利，否则第二个人无论怎么选$v_1$，$v_1$一定被$M$许配（$M$是最大匹配，否则$M\cup \{v_0v_1\}$是更大匹配），那么按照必要性中的策略选取，第一个人一定能最后选到一个顶点，否则存在关于$M$的可增广轨道，与$M$是最大匹配矛盾！

T5

（1）对于$K_{2n}$，考虑$v_1,\cdots,v_{2n-1}$这$2n-1$个顶点构成的正多边形，$v_0$为中心，对于任意$i$，取$v_0v_i$以及与之垂直的边$v_{i+k}v_{i-k}$，下标是模$2n-1$意义下的，这样构成一个$1$度因子$H_i$，且$i\neq j$时，$H_i$和$H_j$无公共边，从而完成$1$度因子分解。

对于$K_{n,n}$​，不妨$X,Y$​中的顶点标记为$1,\cdots,n$​，则将$X$​中的$i$​与$Y$​中的$i+k$​相连（模$n$意义下​）可得一个$1$度因子分解$H_k$，$k$取遍$0,\cdots,n-1$即完成$1$​度因子分解。

（2）在$Petesen$图中任去一个$1$度因子，剩下的图构成两个顶点数为$5$的圈，不可能再有$1$度因子，从而不能$1$度因子分解。

T6

考虑进行染色（黑白相间），将对角上两个方格染成黑色，然后黑色相邻的都是白色，白色相邻的都是黑色。记白色的为$X$，黑色的为$Y$，若两方格相邻，则有边，于是上述去掉两个对角方块的正方形对应于一个二分图，原题等价于没有完备匹配。而$|X|=|Y|+2$，显然不存在完备匹配。

T7

反例：$K_{2n+1}$，无完备匹配。

T8

（1）由推论$5.1$，每个$k$次正则二分图都有完备匹配$M_1$，考虑$G-M_1$，易知是$k-1$次正则二分图，有完备匹配$M_2$，可进行$k$次，所得$M_1\cup \cdots \cup M_k$即为$1$度因子分解。

（2）由定理$6.1$​，$2k$​次正则图是$Euler$​图，考虑$Euler$​回路$C$​，现由$G$​构作二分图$G’$​，$X=\{x_1,\cdots,x_{\nu}\}，Y=\{y_1,\cdots,y_{\nu}\}$​，$x_i,y_j$​有边相连当且仅当在$C$​​中顺次相连，于是$G’$​是$k$​​次正则二分图，由$(1)$​知可$1$​度因子分解，而事实上，$G’$​中的$1$​度因子对应于$G$​中的$2$​度因子，故$G$​可$2$​度因子分解。

T9

构作二分图$G$​，$X,Y$​为行线集和列线集，两顶点有边当且仅当两条线含有公共的$1$​，易知每个$1$​事实上可以对应$G$​中的唯一一条边，则包含所有$1$​的线集是一个覆盖，没有两个$1$​​在同一条线上的$1$​的集合对应与$G$​​的一个匹配，由定理$5.2$​，立得。

T10

对$k$​​作归纳。如上题构作二分图。

$1.k=1$时显然成立

$2.$​假设$k$​时成立，考虑$k+1$​时情形，易知最多有$m$​列有$k+1$​个$1$​，令$P_{k+1}$为$m\times n$阶$0-1$矩阵，每行恰有$1$个1，每列中$1$的个数不超过$1$个，但是又$k+1$个$1$的那些列一定有$1$​，从而$A-P_{k+1}$降至$k$​时情形，由归纳假设知成立。

T11

令$B=X-S$，$|X|-\underset{S\subset X}{max}(|S|-|N(S)|)=\underset{S\subset X}{min}(|X|-|S|+|N(S)|)=\underset{B\subseteq X}{min}(|B|+|N(X-B)|)$​，易知$B\cup N(X-B)$是$G$​的一个覆盖，且$G$​的任一最小覆盖都可以写成这样的形式，​​则由定理$5.2$​知成立。

T12

由上题可知，$\alpha(G)=|X|-\underset{S\subseteq X}{max}(|S|-|N(S)|)$​（注意上题证明过程显然对$S\subseteq X$​同样成立），而存在将$X$​中顶点都许配的顶点当且仅当$\underset{S\subseteq X}{max}(|S|-|N(S)|)=0$​​​，即$\forall S\subseteq X,|N(S)|\geq |S|$。

T13

设$G$是二分图，若$\nu$为偶数，则增加边变成图$H$使得$H[Y]$为完全图；若$\nu$为奇数，则增加一个孤立点$y$，再增加新边得到$H$，使得$H[Y\cup\{y\}]$为完全图，易知$G$中存在将$X$中点都许配的匹配等价于$H$中存在完备匹配，则此时$Hall$定理等价于$H$中存在完备匹配的充要条件是$\forall S\subseteq X,|N_H(S)|\geq |S|$​。

必要性：$\forall S\subseteq X$，令$S_1=N_H(S)$，则$H-S_1$中$S$的每个顶点均是一个奇片，由$Tutte$定理，$|N_H(S)|=|S_1|\geq o(H-S_1)\geq |S|$。

充分性：取$|S|=1$​，由$N_H(S)\geq|S| = 1$​，知$X$​​中无孤立点。$\forall (S = S_1+S_2’)\subseteq V,S_1\subseteq X,S_2’\subseteq Y$​​。令$S_2\subseteq X$​是满足$N_H(S_2)\subseteq S_2’\subseteq Y$​的最大点集。则$|S_2’|\geq|N_H(S_2)|\geq|S_2|$​。$H-S_2’$​共$|S_2|+1$​个连通片，其中$S_2$​个连通片是孤立点，记剩下的那个分支为$H_1$​，$H_1-S_1$​​显然仍然是一个连通片，注意到$S_1,S_2$​可能相交，若$H_1-S_1$​是奇片，则$o(H-S)\leq1+|S_2|$​；若$H_1-S_1$是偶片，则$o(H-S)\leq |S_2|$​。下面分类讨论。

$1.S_1\neq\phi，|S|=|S_1|+|S_2’|\geq 1+|S_2|\geq o(H-S)$​​​

$2.S_1=\phi$​​，若$|S_2’|\geq 1+|S_2|，|S|=|S_1|+|S_2’|=|S_2’|\geq 1+|S_2|\geq o(H-S)$​​，成立；若$|S_2’|=|S_2|$​​，由于$\nu(H)$​​为偶数，则$H_1-S_1$​​是偶片，$|S|=|S_1|+|S_2’|= |S_2’| =|S_2|\geq o(H-S)$

综上，$\forall S\subseteq V(H),o(H-S)\leq|S|$，由$Tutte$定理知，$H$存在完备匹配。

T14

$k=1$显然成立，下考虑$k\geq2$，令$S$是$V$的非空子集，记$G_1,\cdots,G_n$是$G-S$​的所有奇片，$o(G-S)=n$，令$m_i$是$G_i$和$S$​间的相连边数，有$m_i=\sum_{v\in V(G_i)}d(v)-2\epsilon(G_1)=k|V(G_i)|-2\epsilon(G_i)$，则$m_i$与$k$同奇偶，与$k-1$奇偶性相反。由于$G$是$k-1$边连通，故$G_i$和$S$​之间至少有$k-1$条边连通，则$m_i\geq k$，于是$o(G-S)=n\leq\frac{1}{k}\sum_{i=1}^nm_i\leq\frac{1}{k}\sum_{v\in S}d(v)=|S|$。而$S=\phi$时，由于$\nu(G)$是偶数，显然有$o(G-S)=o(G)=0\leq|S|=0$。从而由$Tutte$定理，存在完备匹配。​

T15

必要性：由$Tutte$定理，$o(T-v)\leq|\{v\}|=1$，且$\nu(T)$是偶数，故$\nu(T-v)$是奇数，从而$o(T-v)\geq1$，进而$o(T-v)=1$。

充分性：由于$\forall v\in V,o(T-v)=1$​​，即$T-v$​中存在唯一奇片$C$​，令$v$​与$C$中关联的边$e(v)=vu$，​其由$v$唯一确定，易知$e(u)=vu$，从而$M={e(v)|v\in T}$即为一个完备匹配。

T16

找能将二分图中$X$的顶点都许配的匹配。$abcef$

T17

function [assignment,cost] = munkres(costMat)
% MUNKRES   Munkres Assign Algorithm 
%
% [ASSIGN,COST] = munkres(COSTMAT) returns the optimal assignment in ASSIGN
% with the minimum COST based on the assignment problem represented by the
% COSTMAT, where the (i,j)th element represents the cost to assign the jth
% job to the ith worker.
%
% This is vectorized implementation of the algorithm. It is the fastest
% among all Matlab implementations of the algorithm.
% Examples
% Example 1: a 5 x 5 example
%{
[assignment,cost] = munkres(magic(5));
[assignedrows,dum]=find(assignment);
disp(assignedrows'); % 3 2 1 5 4
disp(cost); %15
%}
% Example 2: 400 x 400 random data
%{
n=400;
A=rand(n);
tic
[a,b]=munkres(A);
toc                 % about 6 seconds 
%}
% Reference:
% "Munkres' Assignment Algorithm, Modified for Rectangular Matrices", 
% http://csclab.murraystate.edu/bob.pilgrim/445/munkres.html
% version 1.0 by Yi Cao at Cranfield University on 17th June 2008
assignment = false(size(costMat));
cost = 0;
costMat(costMat~=costMat)=Inf;
validMat = costMat<Inf;
validCol = any(validMat);
validRow = any(validMat,2);
nRows = sum(validRow);
nCols = sum(validCol);
n = max(nRows,nCols);
if ~n
    return
end
    
dMat = zeros(n);
dMat(1:nRows,1:nCols) = costMat(validRow,validCol);
%*************************************************
% Munkres' Assignment Algorithm starts here
%*************************************************
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
%   STEP 1: Subtract the row minimum from each row.
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
 dMat = bsxfun(@minus, dMat, min(dMat,[],2));
%**************************************************************************  
%   STEP 2: Find a zero of dMat. If there are no starred zeros in its
%           column or row start the zero. Repeat for each zero
%**************************************************************************
zP = ~dMat;
starZ = false(n);
while any(zP(:))
    [r,c]=find(zP,1);
    starZ(r,c)=true;
    zP(r,:)=false;
    zP(:,c)=false;
end
while 1
%**************************************************************************
%   STEP 3: Cover each column with a starred zero. If all the columns are
%           covered then the matching is maximum
%**************************************************************************
    primeZ = false(n);
    coverColumn = any(starZ);
    if ~any(~coverColumn)
        break
    end
    coverRow = false(n,1);
    while 1
        %**************************************************************************
        %   STEP 4: Find a noncovered zero and prime it.  If there is no starred
        %           zero in the row containing this primed zero, Go to Step 5.  
        %           Otherwise, cover this row and uncover the column containing 
        %           the starred zero. Continue in this manner until there are no 
        %           uncovered zeros left. Save the smallest uncovered value and 
        %           Go to Step 6.
        %**************************************************************************
        zP(:) = false;
        zP(~coverRow,~coverColumn) = ~dMat(~coverRow,~coverColumn);
        Step = 6;
        while any(any(zP(~coverRow,~coverColumn)))
            [uZr,uZc] = find(zP,1);
            primeZ(uZr,uZc) = true;
            stz = starZ(uZr,:);
            if ~any(stz)
                Step = 5;
                break;
            end
            coverRow(uZr) = true;
            coverColumn(stz) = false;
            zP(uZr,:) = false;
            zP(~coverRow,stz) = ~dMat(~coverRow,stz);
        end
        if Step == 6
            % *************************************************************************
            % STEP 6: Add the minimum uncovered value to every element of each covered
            %         row, and subtract it from every element of each uncovered column.
            %         Return to Step 4 without altering any stars, primes, or covered lines.
            %**************************************************************************
            M=dMat(~coverRow,~coverColumn);
            minval=min(min(M));
            if minval==inf
                return
            end
            dMat(coverRow,coverColumn)=dMat(coverRow,coverColumn)+minval;
            dMat(~coverRow,~coverColumn)=M-minval;
        else
            break
        end
    end
    %**************************************************************************
    % STEP 5:
    %  Construct a series of alternating primed and starred zeros as
    %  follows:
    %  Let Z0 represent the uncovered primed zero found in Step 4.
    %  Let Z1 denote the starred zero in the column of Z0 (if any).
    %  Let Z2 denote the primed zero in the row of Z1 (there will always
    %  be one).  Continue until the series terminates at a primed zero
    %  that has no starred zero in its column.  Unstar each starred
    %  zero of the series, star each primed zero of the series, erase
    %  all primes and uncover every line in the matrix.  Return to Step 3.
    %**************************************************************************
    rowZ1 = starZ(:,uZc);
    starZ(uZr,uZc)=true;
    while any(rowZ1)
        starZ(rowZ1,uZc)=false;
        uZc = primeZ(rowZ1,:);
        uZr = rowZ1;
        rowZ1 = starZ(:,uZc);
        starZ(uZr,uZc)=true;
    end
end
% Cost of assignment
assignment(validRow,validCol) = starZ(1:nRows,1:nCols);
cost = sum(costMat(assignment));
end

T18

如$T9$​构作二分图，两顶点间有边当且仅当，这一行一列交叉的格子是所选的$16$个格子之一。易知此图为$2$​次正则二分图，有完备匹配$M_1$，在$M_1$所对应的格子上放白子，剩下的放黑子即可。

T19

由于$\alpha_l=\underset{x_i\in S,y_j\notin T}{min}\{l(x_i)+l(y_j)-w(x_iy_j)\}\leq l(x)+l(y)-w(xy)(\forall x\in S,y\notin T)$​​​​​，注意可行定标的修改方式，分类讨论

$1.x\notin S,y\in T$，$\widehat{l}(x)+\widehat{l}(y)= l(x)+l(y)+\alpha(l)\geq l(x)+l(y)\geq w(xy)$​​​

$2.x\notin S,y\notin T,\widehat{l}(x)+\widehat{l}(y)= l(x)+l(y)\geq l(x)+l(y)\geq w(xy)$​

$3.x\in S,y\in T$​，$\widehat{l}(x)+\widehat{l}(y)=l(x)+l(y)-\alpha_l+\alpha_l=l(x)+l(y)\geq w(xy)$​

$4.x\in S,Y\notin T$​​​的情形，$\widehat{l}(x)+\widehat{l}(y)=l(x)+l(y)-\alpha_l\geq l(x)+l(y)-(l(x)+l(y)-w(xy))\geq w(xy)$​​​。

综上，仍然是可行顶标。​

T20

称$l(x_i)+l(y_j)=w(x_iy_j)$​的边$x_iy_j$​​为可行边。以下说明，每次修改顶标后，至少会增加一条可行边，而由匈牙利算法（结合交错树算法）去寻找新的最大匹配时，原来的$T$​显然还在，而且至少还会再增广进一个顶点。

如上题四种分类。

对于$1$中情形，显然原来不可能有可行边，否则$x\in S$，修改之后也不会有。

对于$2，3$中情形，显然不改变可行边。

对于$4$中情形，显然原来不会有可行边，否则$y\in T$​​，但是一定存在$x_iy_j$使得$\alpha_l=l(x_i)+l(y_j)-w(x_iy_j)$​，对于这个$x_iy_j$就是一条新的可行边，且至少有一条。
